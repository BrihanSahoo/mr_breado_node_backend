# V59 - Outlet Order Routing and Audit

- Orders created from a selected outlet are stored with that outlet id.
- Outlet inventory is validated and decremented during order placement.
- Outlet manager login receives only orders for its assigned outlet.
- Admin can view complete order history for each outlet and all outlets.
- Outlet order audit and assignment tables are created automatically.
- Razorpay v22/v26 direct create-order flow is unchanged.
